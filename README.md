

#### 1. Clone the repo:

    ```sh
    git clone https://github.com/dwyl/hapi-register-example.git
    ```
#### 2. Install *Dependencies* from NPM

    ```sh
    npm install
    ```

#### 3. Run the Server

    ```sh
    npm start
    ```

    That's it.
    Now, hack to heart's content!
